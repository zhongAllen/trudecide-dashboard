## 一、项目概览

Trudecide 股票版是一个基于**宏观常识判断**的股票交易策略 AI 系统，核心理念是 Top-Down 四层漏斗选股：

```
Layer 1 宏观状态矩阵（CN + US）
    ↓
Layer 2 板块轮动（热力图 + 排行榜 + 迷你K线）
    ↓
Layer 3 个股精选（GARP 评分 + 财务趋势）
    ↓
Layer 4 公司画布（Company Canvas，深度尽调）
```

**技术栈**：Vite + React + TypeScript + TailwindCSS + Supabase（PostgreSQL）  
**部署地址**：`http://3000-ifzav7aas4ipl6d2aysjv-ce6049a2.sg1.manus.computer`  
**知识库**：`/knowledge`（含 meta 标签，AI 必读）  
**数据资产看板**：`/admin/data`（唯一可信数据底子来源）

## 四、前端功能现状（截至 2026-03-03）

### 4.1 Top-Down 四层漏斗（`/topdown`）

| 层级 | 功能 | 数据状态 |
|---|---|---|
| Layer 1 宏观矩阵 | 5维度×3周期评分矩阵，单元格可展开因子详情 | **✓ 真实数据**（CN + US 均已接入） |
| Layer 2 板块轮动 | 热力图 + 涨跌排行 + 迷你K线 | **✓ 真实数据**（dc 系统板块） |
| Layer 3 个股精选 | GARP 评分 + 财务趋势 + 资金流 | **✓ 真实数据**（含 PE null 方案 A） |
| Layer 4 公司画布 | 公司基本面深度展示 | **⚠ Mock 数据**（4张新表数据待采集后接入） |

### 4.2 PE(TTM) null 处理规则（方案 A，已实施）

- **亏损股**（ROE < 0 或净利润 < 0）：GARP 估值分项强制给 **0 分**，前端显示红色「亏损」标签
- **盈利但 PE 为 null**（新股等）：用**板块中位数 PE** 代替，前端显示灰色「~XX≈」+ hover 提示
- 规则写入注释，完全透明可追溯

### 4.3 宏观矩阵评分规则

评分规则文档：`docs/macro_matrix_scoring_rules.md`  
每个单元格 score = Σ(因子 raw_score × 权重)，规则基于金融常识分段函数，展开面板显示因子贡献表。

### 4.4 北向资金替代方案

`north_net_flow`（净流入）已废弃（REQ-119），新口径未采集。当前替代：
- `north_daily_turnover`：北向当日成交额（反映外资活跃度）
- `north_turnover_ratio_daily`：北向占全A比例（反映外资参与度）

### 4.5 M2 驾驶舱（`/`）

主页面，含持仓模块、指标看板、双地图、新闻快览。已完成（REQ-137~141）。
