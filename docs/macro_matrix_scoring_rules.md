# 宏观状态矩阵评分规则 v1 (REQ-159)

本文档定义了 Top-Down 选股模型中，Layer 1 宏观状态矩阵的评分计算规则。所有评分均基于 `indicator_values` 表中的真实数据，通过可解释的、基于金融常识的规则体系生成，确保了评分的透明度和可追溯性。

---

## 一、核心设计原则

1.  **可解释性 (Explainable)**：每个单元格的最终得分（0-100）必须能分解为若干个基础指标（因子）的加权贡献，杜绝黑箱模型。
2.  **数据驱动 (Data-Driven)**：所有计算的输入值均来自 Supabase 数据库中的真实、最新宏观指标数据。
3.  **模块化 (Modular)**：评分规则按「维度-周期」单元格进行模块化组织，每个基础指标的评分函数（`value` -> `raw_score`）可复用。
4.  **版本化 (Versioned)**：本文档作为评分规则的唯一信源，任何规则调整都必须更新本文档并记录版本。

---

## 二、通用评分函数

为了将不同单位、不同量纲的宏观指标归一化到 0-100 分的评分区间，我们定义一个通用的分段线性评分函数 `mapToScore(value, thresholds)`。此函数根据预设的阈值将指标的当前值映射为原始分（Raw Score）。

-   **正向指标**（值越大越好，如 PMI、GDP增速）：
    -   `value >= thresholds[4]` → 100分 (极强)
    -   `thresholds[3] <= value < thresholds[4]` → 80分 (偏强)
    -   `thresholds[2] <= value < thresholds[3]` → 60分 (中性偏强)
    -   `thresholds[1] <= value < thresholds[2]` → 40分 (中性偏弱)
    -   `value < thresholds[1]` → 20分 (偏弱)
-   **负向指标**（值越小越好，如失业率）：规则相反，`value <= thresholds[0]` 为100分。

## 三、CN 宏观矩阵评分规则详解

### 维度 1：经济周期位置

**A股相关性：** 正相关

#### 1.1 短期 (3-9个月)

**综合描述：** 基于 PMI、GDP、工业产出、消费等高频指标，判断短期经济景气度。

| 因子 (Indicator) | 权重 (Weight) | 评分规则 (Thresholds for `mapToScore`) | 规则说明 (金融常识) |
| :--- | :---: | :--- | :--- |
| **制造业PMI** (`pmi_mfg`) | 30% | `[48, 49.5, 50.5, 51.5]` | 50为荣枯线。持续高于50.5表示强劲扩张，低于49.5则有收缩压力。 |
| **非制造业PMI** (`pmi_non_mfg`) | 20% | `[49, 50.5, 52, 54]` | 服务业和建筑业景气度的重要补充，通常波动比制造业小，阈值略高。 |
| **GDP同比增速** (`gdp_yoy`) | 25% | `[3.5, 4.5, 5.5, 6.5]` | 政府目标通常在5%左右，高于5.5%为强劲，低于4.5%则增长乏力。 |
| **工业增加值同比** (`industrial_yoy`) | 15% | `[2, 4, 6, 8]` | 工业生产的直接反映，6%以上通常被视为非常健康。 |
| **社消零售总额同比** (`retail_yoy`) | 10% | `[1, 3, 5, 7]` | 反映消费需求，是内需的重要支柱。5%以上增长较为理想。 |

**单元格总分计算公式：**
`Score = (pmi_mfg_score * 0.3) + (pmi_non_mfg_score * 0.2) + (gdp_yoy_score * 0.25) + (industrial_yoy_score * 0.15) + (retail_yoy_score * 0.1)`

#### 1.2 中期 (2-3年)

**综合描述：** 关注投资、出口、就业等影响中期增长动能的因素。

| 因子 (Indicator) | 权重 (Weight) | 评分规则 (Thresholds for `mapToScore`) | 规则说明 (金融常识) |
| :--- | :---: | :--- | :--- |
| **GDP同比增速** (`gdp_yoy`) | 30% | `[3.5, 4.5, 5.5, 6.5]` | 中期增长中枢，权重最高。 |
| **固定资产投资同比** (`fai_yoy`) | 25% | `[2, 4, 6, 8]` | 投资是拉动经济增长的三驾马车之一，反映中期增长潜力。 |
| **出口金额同比(美元计)** (`export_yoy`) | 20% | `[-5, 0, 5, 10]` | 外需情况的直接体现，0以上表示恢复增长。 |
| **城镇调查失业率** (`unemployment_rate`) | 15% | `[5.5, 5.2, 4.8, 4.5]` (负向) | 就业是民生之本，也是经济的最终体现。低于5%为健康水平。 |
| **进口金额同比(美元计)** (`import_yoy`) | 10% | `[-5, 0, 5, 10]` | 反映内需和生产活动的活跃度。 |

**单元格总分计算公式：**
`Score = (gdp_yoy_score * 0.3) + (fai_yoy_score * 0.25) + (export_yoy_score * 0.2) + (unemployment_rate_score * 0.15) + (import_yoy_score * 0.1)`

#### 1.3 长期 (5-10年)

**综合描述：** 关注潜在增速和结构性问题。

| 因子 (Indicator) | 权重 (Weight) | 评分规则 (Thresholds for `mapToScore`) | 规则说明 (金融常识) |
| :--- | :---: | :--- | :--- |
| **GDP潜在增速趋势** | 60% | *定性判断* | 基于主流经济学家的长期预测，目前普遍认为在4-5%区间，因此给予中性偏弱的45分基础分。 |
| **劳动力市场结构** | 40% | *定性判断* | 考虑人口老龄化、总和生育率等长期结构性因素，对劳动力供给构成压力，给予偏弱的35分基础分。 |

**单元格总分计算公式：**
`Score = (45 * 0.6) + (35 * 0.4) = 41` (长期偏弱)`

### 维度 2：货币政策信号

**A股相关性：** 正相关

#### 2.1 短期 (3-9个月)

**综合描述：** 关注市场利率和央行公开市场操作，判断短期流动性松紧。

| 因子 (Indicator) | 权重 (Weight) | 评分规则 (Thresholds for `mapToScore`) | 规则说明 (金融常识) |
| :--- | :---: | :--- | :--- |
| **1年期LPR** (`lpr_1y`) | 30% | `[3.8, 3.6, 3.4, 3.2]` (负向) | LPR是贷款市场报价利率，直接影响企业融资成本。越低越宽松。 |
| **DR007** (`dr007`) | 25% | `[2.2, 2.0, 1.8, 1.6]` (负向) | 银行间质押式回购利率，是观察银行间流动性的核心指标。低于政策利率（如MLF）越多越宽松。 |
| **10年期国债收益率** (`bond_10y`) | 25% | `[3.0, 2.8, 2.6, 2.4]` (负向) | 无风险利率的锚，反映市场对长期经济和通胀的预期。 |
| **5年期LPR** (`lpr_5y`) | 20% | `[4.5, 4.2, 3.9, 3.6]` (负向) | 主要影响房贷利率，对地产和银行板块有重要影响。 |

**单元格总分计算公式：**
`Score = (lpr_1y_score * 0.3) + (dr007_score * 0.25) + (bond_10y_score * 0.25) + (lpr_5y_score * 0.2)`

#### 2.2 中期 (2-3年)

**综合描述：** 关注货币政策立场和汇率约束。

| 因子 (Indicator) | 权重 (Weight) | 评分规则 (Thresholds for `mapToScore`) | 规则说明 (金融常识) |
| :--- | :---: | :--- | :--- |
| **10年期国债收益率** (`bond_10y`) | 40% | `[3.0, 2.8, 2.6, 2.4]` (负向) | 中期利率中枢的体现。 |
| **1年期LPR** (`lpr_1y`) | 35% | `[3.8, 3.6, 3.4, 3.2]` (负向) | 反映中期信贷政策方向。 |
| **人民币兑美元中间价** (`rmb_usd`) | 25% | `[7.3, 7.1, 6.9, 6.7]` (负向) | 汇率对货币政策独立性构成约束。汇率贬值压力越大，宽松空间越小。 |

**单元格总分计算公式：**
`Score = (bond_10y_score * 0.4) + (lpr_1y_score * 0.35) + (rmb_usd_score * 0.25)`

#### 2.3 长期 (5-10年)

**综合描述：** 关注长期利率中枢和结构性约束。

| 因子 (Indicator) | 权重 (Weight) | 评分规则 (Thresholds for `mapToScore`) | 规则说明 (金融常识) |
| :--- | :---: | :--- | :--- |
| **10年期国债收益率趋势** | 60% | *定性判断* | 长期看，随经济增速放缓，无风险利率中枢下行是大概率事件，给予中性偏多60分。 |
| **债务/GDP比率** | 40% | *定性判断* | 宏观杠杆率高位运行，限制了长期大幅宽松的空间，给予中性偏弱40分。 |

**单元格总分计算公式：**
`Score = (60 * 0.6) + (40 * 0.4) = 52` (长期中性)

### 维度 3：政策底确认

**A股相关性：** 正相关

#### 3.1 短期 (3-9个月)

**综合描述：** 关注信贷和社融数据，判断信用扩张力度。

| 因子 (Indicator) | 权重 (Weight) | 评分规则 (Thresholds for `mapToScore`) | 规则说明 (金融常识) |
| :--- | :---: | :--- | :--- |
| **M2同比增速** (`m2_yoy`) | 30% | `[7, 8, 9, 10]` | M2增速需与名义GDP增速匹配，8-9%是当前较合理的区间。 |
| **社融存量同比** (`social_finance_yoy`) | 30% | `[7, 8, 9, 10]` | 社融是更广义的货币供应，更能反映实体经济的融资需求。 |
| **新增人民币贷款** (`new_loans`) | 25% | `[1.0, 1.5, 2.0, 2.5]` (万亿) | 月度信贷投放的直接体现，季节性强，需结合历史同期比较。 |
| **社融增量** (`social_finance_new`) | 15% | `[1.5, 2.5, 3.5, 4.5]` (万亿) | 同样季节性强，关注其同比多增或少增的趋势。 |

**单元格总分计算公式：**
`Score = (m2_yoy_score * 0.3) + (social_finance_yoy_score * 0.3) + (new_loans_score * 0.25) + (social_finance_new_score * 0.15)`

### 维度 4：流动性环境

**A股相关性：** 正相关

#### 4.1 短期 (3-9个月)

**综合描述：** 关注市场成交、杠杆资金和外资动向。

| 因子 (Indicator) | 权重 (Weight) | 评分规则 (Thresholds for `mapToScore`) | 规则说明 (金融常识) |
| :--- | :---: | :--- | :--- |
| **全A日成交额** (`total_market_turnover`) | 30% | `[0.7, 0.9, 1.1, 1.3]` (万亿) | 市场活跃度的最直接指标。万亿以上为活跃，低于7000亿为低迷。 |
| **M2同比增速** (`m2_yoy`) | 25% | `[7, 8, 9, 10]` | 宏观流动性的总阀门。 |
| **北向成交额** (`north_daily_turnover`) | 25% | `[1000, 1200, 1500, 1800]` (亿) | 外资情绪的“温度计”。持续高于1500亿表明外资积极参与。 |
| **两市融资余额** (`margin_balance_total`) | 20% | `[1.4, 1.5, 1.6, 1.7]` (万亿) | 杠杆资金的活跃度，反映市场风险偏好。 |

**单元格总分计算公式：**
`Score = (turnover_score * 0.3) + (m2_yoy_score * 0.25) + (north_turnover_score * 0.25) + (margin_balance_score * 0.2)`

**A股相关性：** 正相关

**综合描述：** 关注汇率、贸易和全球风险偏好。

| 因子 (Indicator) | 权重 (Weight) | 评分规则 (Thresholds for `mapToScore`) | 规则说明 (金融常识) |
